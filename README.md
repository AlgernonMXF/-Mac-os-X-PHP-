# -Mac-os-X-PHP-
[后端开发入门] 配置PHP开发环境：包括homebrew，cockscrew, git, PHPstorm, sourceTree下载安装

## 写在前面
	软件安装及使用过程中需要连接到外网，所以需要提前做好翻墙工作
	
	具体方法包括：
	1 使用VPN(如Lantern)
	2 租用(阿里云)服务器并选择美国境内站点，下载Shimo软件使用PPTP协议连接
	3 直接连接公司已经配置好的路由器(记得感谢借你使用路由器的大神)
